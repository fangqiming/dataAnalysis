/trade/search_gain

获取收益信息

/trade/search_gain get

get

get /trade/search_gain

  • Request
  • Response

Query Parameters

  • pageSize: required (integer)

    页大小

  • pageNo: required (integer)

    第几页

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
   "code": 0,
   "data": {
     "entities": [
       {
         "id": 6,      //主键
         "date": "2017-04-06",   //日期
         "stock": 779.85,        //股票金额
         "balance": 297.43,      //账户余额
         "cover": 114.09,        //做空余额 （为负数表示收益）
         "gain": -0.0004         //收益率
       }
     ],
     "total": 6
   }
 }
